# SHG TikTok Shop 菲律宾利润计算器

纯 HTML、CSS、JavaScript 的网页版本利润计算器，无需后端。页面语言为中文，数据会按“菲律宾本土发货”和“跨境发货”两套模式分别保存到浏览器 `localStorage`。

## 如何本地打开

1. 下载或克隆本仓库。
2. 确保目录中包含：
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
3. 直接用浏览器双击打开 `index.html`。
4. 也可以在项目目录启动任意静态服务器，例如：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 如何使用本土发货模式

1. 打开页面后默认进入“菲律宾本土发货”模式。
2. 在【买家端】填写商品原价、商品活动折扣、Seller coupon、TikTok Shop coupon、买家支付运费。
3. TikTok Shop coupon 只影响买家前台小计和买家最终支付金额，不会从卖家收入或成本中直接扣除。
4. 在【卖家结算端】选择商家总折扣计算方式：
   - 自动：商家总折扣 = 商品活动折扣 + Seller coupon。
   - 手动：直接填写商家总折扣；手动优先，用于匹配真实 TikTok Shop 后台结算单。
5. 本土模式卖家收入基数为“商家折扣后小计 = 商品原价 - 商家总折扣”。
6. 填写本土履约成本，包括采购 RMB、汇率、国内物流、国际头程、本土仓储、仓库操作、尾程、退货/拒收/坏件/库存损耗等。
7. 点击“开始计算”，查看预计结算总金额、最终净利润、利润率、保本原价和建议原价。

## 如何使用跨境发货模式

1. 点击顶部“跨境发货”。
2. 填写价格输入、商家总折扣和商家折扣后小计相关数据。
3. 填写跨境成本，包括采购成本 RMB、汇率、国内发货到集运仓、国内包装、国际物流 / 跨境履约成本、售后损耗、坏件补发、其他履约成本。
4. 跨境平台费用默认全部关闭，因为目前没有真实跨境结算单；请根据后续拿到的真实账单自行启用并填写费率或固定金额。
5. 点击“开始计算”查看跨境预计结算金额、平台费用总额、提现手续费、最终净利润、利润率、保本原价和建议原价。

## 如何修改费率

每项平台费用都支持：

- 勾选 / 取消勾选：控制费用是否启用。
- 计算方式：选择“按比例 %”或“固定 PHP”。
- 费率 / 金额：按比例时填写百分比，固定时填写 PHP 金额。
- 手动固定金额：如果填写，将优先使用该固定金额，适合对照真实结算单。

本土模式默认内置真实结算单逻辑下的默认费率：交易手续费 2.24%、TikTok Shop 佣金 7.2%、达人 / 联盟佣金 10%、SFP 5.5%、成长服务费 1%、订单处理 3 PHP、物流保障 2 PHP 等。超级优惠券、促销抽点、秒杀、运费补贴可按活动情况启用。

提现手续费支持按商家折扣后小计、按平台扣费后结算金额或关闭；默认按平台扣费后结算金额计算。

## 如何用案例 A、案例 B 做结算单校验

1. 保持在“菲律宾本土发货”模式。
2. 在“本土结算单校验案例”下拉框选择“案例 A”或“案例 B”。
3. 页面会自动填充商品原价、商家总折扣和真实固定平台费用金额。
4. 校验案例使用真实固定金额，不按默认费率反推。
5. 页面会显示：
   - “结算单校验通过”或“结算单校验不一致”
   - 后台真实结算金额
   - 计算结算金额
   - 差额
   - 真实平台费用与计算平台费用

内置校验目标：

- 案例 A：商品原价 550 PHP，商家总折扣 225 PHP，平台费用总额 87.43 PHP，预计结算总金额 237.57 PHP。
- 案例 B：商品原价 550 PHP，商家总折扣 235 PHP，平台费用总额 104.70 PHP，预计结算总金额 210.30 PHP。

## 跨境模式等待真实结算单校准的费率

跨境模式以下项目默认关闭，等待后续真实结算单校准：

- 平台佣金
- 交易手续费
- 达人佣金
- SFP 服务费
- 促销活动服务费
- 秒杀服务费
- 提现手续费
- 其他固定费用
- 其他比例费用
