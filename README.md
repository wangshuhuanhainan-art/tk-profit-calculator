# 超级 TikTok Shop 菲律宾利润计算器

纯 HTML、CSS、JavaScript 的网页版本利润计算器，无需后端。页面语言为中文，数据会按“菲律宾本土发货”和“跨境发货”两套模式分别保存到浏览器 `localStorage`。

## 如何本地打开

1. 下载或克隆本仓库。
2. 确保目录中包含：
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
3. 直接用浏览器双击打开 `index.html`。
4. 也可以在项目目录启动任意静态服务器，例如：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 如何使用本土发货模式

1. 打开页面后默认进入“菲律宾本土发货”模式。
2. 在【买家端】填写商品原价、商品活动折扣、Seller coupon、TikTok Shop coupon、买家支付运费。
3. TikTok Shop coupon 只影响买家前台小计和买家最终支付金额，不会从卖家收入或成本中直接扣除。
4. 平台费用端会按“商家优惠后商品金额 = 商品原价 - 商家活动折扣 - Seller Coupon”自动计算。
5. 填写本土履约成本，包括商品采购成本 RMB、人民币兑 PHP 汇率、国内发货到集运仓成本 RMB、国际物流成本 RMB、海外仓出库费用 RMB / 单和其他成本 PHP；买家支付运费和商家承担运费保留在订单结算逻辑中。海外仓出库费用会按“RMB / 单 × 人民币兑 PHP 汇率”换算为 PHP 后计入产品总成本。
6. 点击“开始计算”，查看预计平台结算金额、风险调整后卖家结算金额、最终利润、利润率、保本售价和目标利润售价。

## 如何使用跨境发货模式

1. 点击顶部“跨境发货”。
2. 填写价格、优惠、买家运费和成本相关数据。
3. 填写跨境成本，包括商品采购成本 RMB、人民币兑 PHP 汇率、国内发货到集运仓成本 RMB、海外仓出库费用 RMB / 单、国际物流 / 跨境履约成本 PHP、其他成本 PHP。海外仓出库费用会按“RMB / 单 × 人民币兑 PHP 汇率”换算为 PHP 后计入产品总成本。
4. 跨境平台费用默认全部关闭，因为目前没有真实跨境结算单；请根据后续拿到的真实账单自行启用并填写费率或固定金额。
5. 点击“开始计算”查看跨境预计结算金额、平台费用总额、最终净利润、利润率、保本原价和建议原价。

## 如何修改费率

每项平台费用都支持：

- 勾选 / 取消勾选：控制费用是否启用。
- 计算方式：选择“按比例 %”或“固定 PHP”。
- 费率 / 金额：按比例时填写百分比，固定时填写 PHP 金额。
- 计费基数：支持商家优惠后商品金额、买家最终付款金额、手动输入金额或固定 PHP 金额。
- 手动基数 / 实际金额：比例费用选择手动基数时作为计费基数；固定费用可作为实际固定扣费。

本土模式默认内置校准后的费率：TikTok Shop 佣金 7.2%、交易手续费 2.24%（按买家最终付款金额）、SFP 5.5%、成长服务费 1%、订单处理 3 PHP、达人 / 联盟佣金 10%、超级优惠券服务费 4%。秒杀、商家活动扣点、物流保障服务费、其他平台费用可按活动情况启用。

卖家结算端保留总标题，并拆分为两个独立子模块：【广告费用】只包含广告启用开关、广告输入方式、广告费 RMB / 费率以及 RMB/PHP 结果；【风险预留与其他卖家费用】包含退货率、拒收率、取消率、商家承担运费 PHP、其他卖家结算端费用 PHP以及对应风险预留和调整后结算结果。广告开关只控制广告费用，不影响风险预留、商家承担运费或其他卖家费用。

## 真实订单复盘

真实订单复盘模式可录入平台账单实际金额、实际达人佣金、实际广告费、实际商家运费与实际风险损失，用于对比计算器预计结算金额和真实结算结果。

## 跨境模式等待真实结算单校准的费率

跨境模式以下项目默认关闭，等待后续真实结算单校准：

- 平台佣金
- 交易手续费
- 达人佣金
- SFP 服务费
- 促销活动服务费
- 秒杀服务费
- 其他固定费用
- 其他比例费用


## 2026 校准说明

- 商品原价 = 商品标价。
- 商家优惠后商品金额 = 商品原价 - 商家活动折扣 - Seller Coupon。
- 买家最终付款金额 = 商家优惠后商品金额 + 买家支付运费。
- 平台交易手续费默认按买家最终付款金额计算；其他默认比例费用按商家优惠后商品金额计算。
- 达人 / 联盟佣金默认开启并按 10% 预估；真实订单复盘可填写实际费率或实际金额临时覆盖，不影响默认值。
- 超级优惠券服务费默认开启并按 4% 预估；真实订单复盘可临时关闭、修改或输入实际金额，不影响默认值。
- 广告费用位于卖家结算端的独立【广告费用】子模块，按 RMB 输入并使用“人民币兑 PHP 汇率”换算为 PHP 后扣除；广告开关只影响广告相关字段和结果。
- 退货率、拒收率、取消率位于卖家结算端的独立【风险预留与其他卖家费用】子模块，总风险率最高按 100% 计算，与广告费用独立。
- TikTok / Platform Coupon 保持“平台承担，仅展示”定位，不作为卖家费用扣减，不进入产品成本，也不直接影响最终利润。
- 如浏览器中存在历史旧成本字段，本页面会保留历史值并提示用户重新填写“海外仓出库费用 RMB / 单”；历史旧值不会与新出库费用同时计入产品总成本。

## PWA 安装与离线使用

本工具已升级为可安装的 PWA 应用，应用名称为“牛逼clas计算器”，短名称为“牛逼clas”。页面主标题仍保持“超级 TikTok Shop 菲律宾利润计算器”。

### 手机如何添加到桌面

- 安卓 Chrome：打开网页后点击浏览器菜单，选择“添加到主屏幕”。如果浏览器显示安装提示，也可以直接点击页面顶部的“安装应用”。
- iPhone Safari：点击底部或顶部的分享按钮，选择“添加到主屏幕”，然后确认添加。

### Windows 电脑如何安装

- Chrome / Edge：打开网页后点击地址栏右侧的安装图标。
- 如果没有看到地址栏图标，可点击浏览器菜单，选择“安装应用”或“应用 > 将此站点作为应用安装”。
- 页面顶部的“安装应用”按钮会优先触发浏览器原生安装流程；不支持时会显示简短安装说明。

### 如何更新应用

- 新版本上线后，Service Worker 会使用新的缓存版本号安装新版资源。
- 页面检测到新版后会显示“发现新版本，点击刷新更新”。点击提示后会切换到新版并刷新页面。
- 如果没有看到提示，关闭所有已打开的本工具页面后重新打开，浏览器也会检查新版本。

### 如何卸载应用

- 手机：长按桌面图标，选择移除、卸载或删除书签式应用。
- Windows Chrome / Edge：打开已安装应用窗口，点击标题栏或菜单中的卸载选项；也可以在浏览器的应用管理页面中移除。

### PWA 离线使用说明

- 首次联网打开后，浏览器会缓存 `index.html`、`style.css`、`script.js`、`manifest.webmanifest` 和应用图标资源。
- 弱网或短时离线时，已缓存页面仍可打开并继续使用已有的计算逻辑。
- 计算方案、默认费率和用户输入仍保存在浏览器 `localStorage` 中，PWA 缓存更新不会清除这些数据。

### 如何避免旧缓存导致页面没有更新

1. 优先点击页面底部出现的“发现新版本，点击刷新更新”。
2. 如果页面仍未更新，请关闭本工具所有标签页和已安装应用窗口后重新打开。
3. 仍异常时，可在浏览器站点设置中仅清除“缓存的图片和文件”或取消注册 Service Worker；不要清除站点数据 / localStorage，否则会删除已保存的计算方案和输入。

> 临时说明：为保证代码评审和 PR 创建过程不受二进制文件限制影响，当前 PWA 图标使用纯文本 SVG 占位文件 `assets/app-icon.svg`。后续可由用户手动上传头像 PNG 到 `assets/app-icon.png`，并将 `manifest.webmanifest`、`index.html` 和 `sw.js` 中的图标路径替换为 PNG 路径。
